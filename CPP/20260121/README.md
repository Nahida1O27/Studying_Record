### 一个为了更改Qt账号的简单powershell脚本
### A simple PowerShell script to switch Qt account

因为我的第一个Qt账号没有申请教育许可，所以我有了第二个账号。这就有了账号切换的需求。  
Because my first Qt account doesn't have educational license, so I take the second. And then I have a demand for account switching.  
实现原理很简单，就是文件的剪贴或移动。  
The principle is very simple； just moving the files.  
脚本并不健壮，没有错误处理等。  
The script is not robust, can't handle error.

Please use carefully, reference is welcome.
